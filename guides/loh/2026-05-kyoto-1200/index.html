結論

最重要 短距離S、終盤加速、コンセントレーション。前脚質は出遅れがかなり重く、序盤の位置取りがそのまま終盤条件に響きます。
おすすめ編成 上位狙いは逃げ2＋先行1。プラチナ4狙いなら逃げ1＋先行2も現実的。差し追込は安定度が低めです。
因子方針 Dreamsで親・祖父母のG1ローテを揃え、短距離赤因子と有効加速の白因子を優先します。

脚質別攻略

逃げ

上位狙い向け
  • 逃げ2以上なら追い比べを意識
  • 単逃げならアンスキ優先
  • 譲れぬ一歩、押せ押せムードを厚くする

先行

最も扱いやすい
  • つぼみ、ほころぶ時が軸
  • 火花散らして、真っ向勝負を優先
  • 中盤速度で3〜5位を確保

差し

上振れ枠
  • ニシノ固有を前に出た保険にする
  • 差し切り体勢、ターニングポイント
  • 順位条件が不安定なのでLoHでは注意

追込

趣味寄り
  • ヒートアップ、切れ味、一足飛び
  • バンブー固有は候補
  • スコア安定目的では優先度低め

本育成で取得すべきスキル優先度

S 最優先

コンセントレーション

全脚質、特に逃げ/先行 スタート事故対策

効果：スタートが得意になり、出遅れる時間が少なくなる。

発動条件：スタート時。京都1200mは序盤の追い越し困難区間と位置取りが重く、前脚質は出遅れがそのまま加速条件の崩れにつながるため最優先。

S 脚質別必須

火花散らして / 真っ向勝負 / 譲れぬ一歩

先行 逃げ 終盤始め加速

火花散らして：レース終盤始めの方で前方にいると加速力がわずかに上がる＜先行・短距離＞。上位は丁々発止。

真っ向勝負：レース終盤始めの方で前方にいると加速力がわずかに上がる＜作戦・先行＞。上位は鍔迫り合い。

譲れぬ一歩：レース終盤始めの方で前方にいると加速力がわずかに上がる＜逃げ・短距離＞。上位はイニシアチブ。

発動条件：京都1200mでは終盤開始が800m付近。終盤始めの加速が有効になりやすいため、先行は火花散らして＋真っ向勝負、逃げは譲れぬ一歩を優先。

A 高優先

地固め / 先手必勝・先駆け / 危険回避 / ポジションセンス

逃げ向け 序盤位置取り

地固め：レース序盤にスキルを多く発動すると加速力がわずかに上がる。上位は盤石の構え。

先手必勝/先駆け：レース序盤で前に行きやすくなる/わずかに前に行きやすくなる＜作戦・逃げ＞。

危険回避：レース序盤に少し囲まれにくくなる＜作戦・逃げ＞。

ポジションセンス：レース序盤にコース取りがうまくなる。

発動条件：いずれも序盤。逃げは先頭争いの負けが致命傷なので、地固め用の緑スキルや序盤スキルとセットで積みます。

A 全脚質

スプリントギア / 押せ押せムード

短距離加速 サブ加速

スプリントギア：レース終盤に加速力がわずかに上がる＜短距離＞。上位はスプリントターボ。

押せ押せムード：レース終盤始めの方で前方にいると加速力がわずかに上がる＜短距離＞。

発動条件：短距離条件。どちらも終盤加速の補助として有効。脚質専用加速を取ったうえで、サブ加速として積む優先度が高いです。

B 後方用

一足飛び / 電光石火

差し/追込 終盤加速

一足飛び：レース終盤に後方で追い抜こうとすると加速力がわずかに上がる＜短距離/マイル＞。上位は電光石火。

電光石火：レース終盤に後方で追い抜こうとすると加速力が上がる＜短距離/マイル＞。

発動条件：終盤に後方で追い抜こうとすること。LoH京都1200mでは差し/追込の安定度が低いため、後方脚質を使う場合のみ採用候補。

B 条件確定

右回り○ / 春ウマ娘○ / 根幹距離○ / 京都レース場○

緑スキル ステータス底上げ

右回り○：右回りコースが少し得意になる。上位に右回りの鬼。

春ウマ娘○：春のレースが少し得意になる。上位に春一番。

根幹距離○：根幹距離、400mの倍数が少し得意になる。京都1200mは該当。

京都レース場○：京都レース場が少し得意になる。上位に淀の申し子。

発動条件：レース条件一致で確定発動。スキルPt効率が良く、地固めの起動にも使えるため、ヒントLvが高ければ優先して取得。

C 余裕枠

短距離直線○ / 短距離コーナー○ / ウマ好み / トップランナー・先頭プライド

速度 ヒントLv優先

短距離直線○：直線で速度がわずかに上がる＜短距離＞。

短距離コーナー○：コーナーで速度がわずかに上がる＜短距離＞。

ウマ好み：近くにウマ娘がたくさんいると速度がわずかに上がる。上位はウママニア。

トップランナー/先頭プライド：レース序盤または中盤に追い抜きや競り合いに強くなる/わずかに強くなる＜逃げ＞。

発動条件：直線、コーナー、近くに複数ウマ娘、逃げの追い抜き/競り合いなど。終盤加速と序盤セットを取った後、ヒントLvが高いものから取得。

スキル効果・条件の確認元： ウマ娘ラボ 京都1200m解説、 コンセントレーション、 真っ向勝負、 火花散らして、 譲れぬ一歩、 地固め、 スプリントギア。

LoHは3人全員の着順でスコアを積みます。1人の爆発力より、3人が大崩れしない編成とスキル構成を優先しましょう。
